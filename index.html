#OurPhilosophy

KISS principle, a design principle noted by the U.S. Navy - "keep it simple, stupid" The KISS principle states that most systems work best if they are kept simple rather than made complicated; therefore simplicity should be a key goal in design, and that unnecessary complexity should be avoided.


Web Jedi lesson

Copy any text and paste in a text editor (Notepad). Turn any text-to-HTML is just to add labels (tags) for start and for end, and save as a new html file.
For example: <html>Done! You are a Web Jedi.</html>
Save as new file text-to-HTML.html and open it with a web browser (Google Chrome).

For real hackers only!!!
<html>
