Overview #OurPhilosophy

KISS principle, a design principle noted by the U.S. Navy - keep it simple, stupid

The KISS principle states that most systems work best if they are kept simple rather than made complicated; therefore simplicity should be a key goal in design, and that unnecessary complexity should be avoided.

Web Jedi HTML lesson

Nobody Likes a Slow Website, so try this:

Copy any text and paste in a text editor (Notepad is the default text editor in Windows or TextEdit is the default text editor in macOS). Turn any text-to-HTML is just to add labels (HTML tags) for start and for end, and save as a new html file.


For example: <html>Done! You are a Web Jedi.</html>
Save as a new file text-to-HTML.html and open it with a web browser (Google Chrome or Mozilla Firefox or Microsoft Edge or Apple Safari or any other favorit browser).

For real hackers only!!!
<html>
<head>As first section is used to instruct machines (search engines, robots-bots, styles, designs, etc.), for example:<title>Title</title></head>
<body>As second section is used to show only the useful content to the visitors (fastest and most important are: <h1>Title</h1>, <h2>Text</h2>, <p>Paragraph</p>, <h3>Description</h3>, <h4>Internal links</h4>, <h5>Note:</h5>, <h6>Copyright</h6>; and slowest are: pictures, videos, any files, etc.)</body>
</html>
Done! You are a Web Jedi master.
